# Secure Password Generator

## Description
This project is a Password Generator developed using HTML, CSS, and JavaScript.

## Features
- Generate secure passwords
- Password length selection
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Copy password
- Show/Hide password
- Password strength meter

## Technologies Used
- HTML5
- CSS3
- JavaScript
